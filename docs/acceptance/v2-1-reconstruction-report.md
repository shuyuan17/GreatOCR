## 质量门禁

质量检查现可报告以下风险：

- `orientation_mismatch`：输出页方向与源 PDF 不一致。
- `asset_missing`：图片、签名或印章资源缺失，或资源路径越出任务目录。
- `possible_english_word_join`：OCR 行边界可能导致英文单词粘连。
- 原有的缺页、关键字段未留痕修改等检查继续保留。

## 安全与外部服务说明

本次验收仅使用本地 fake provider。未执行真实 MinerU 上传，未读取或输出任何 API key，也未向 GitHub 或其他远程仓库上传代码。
