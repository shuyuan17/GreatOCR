## 当前判断

- 后端可以启动，任务 API 已包含文件上传端点
- 前端「新建任务」页面已实现文件选择、上传、自动创建任务、轮询状态
- 前端「任务中心」页面已实现任务列表，每 5 秒自动刷新
- 前端「设置」页面已实现 Provider 列表展示与 API Key 配置指引
- `scripts/serve.py` 一键启动脚本，自动创建 fake-default provider 和数据库
- 真实文件测试已完成：fake provider（离线）和 MinerU（真实 OCR）双流程均跑通
- 真实文件 OCR 质量评级为 **high**，但存在单词粘连、数字格式、未知块类型等问题
