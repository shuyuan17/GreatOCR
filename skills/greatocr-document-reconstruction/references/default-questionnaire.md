# Default Questionnaire

Use these defaults unless the user says otherwise:

- Sensitive file: no
- Source language: auto
- Output mode: editable Word
- Translation: no
- Preserve headers, footers, and page numbers: yes
- Pagination: preserve as much as practical
- Number format: keep original display format
- Date format: keep original display format
- Parser provider: last approved provider
- Normal-mode intermediates: keep JSON and Markdown, clean page cache
- Sensitive-mode intermediates: keep final DOCX and quality report only by default
